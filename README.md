# Pégale al Topo

## Descripción del Proyecto
"Pégale al Topo" es un proyecto educativo diseñado para aprender y practicar conceptos fundamentales de programación a través del desarrollo de un juego interactivo simple. El objetivo es guiar a Gale (el topo) en un entorno básico mientras se exploran mecánicas de juego esenciales.

## Características
- **Mecánicas Básicas de Juego**: Sistema simple de movimiento e interacción
- **Lógica de Programación**: Implementación de conceptos fundamentales como variables, condicionales y bucles
- **Niveles Progresivos**: Estructura sencilla que permite expandir la dificultad gradualmente
- **Código Limpio y Educativo**: Enfoque en buenas prácticas de desarrollo y legibilidad

## Objetivos de Aprendizaje
Este proyecto es ideal para:
- Aprender los fundamentos del desarrollo de juegos
- Practicar lógica de programación
- Entender estructuras de código básicas
- Explorar mecánicas interactivas simples

## Instrucciones de Instalación
Para ejecutar "Pégale al Topo", sigue estos pasos:
1. Clona el repositorio en tu máquina local
2. Instala las dependencias necesarias
3. Ejecuta el archivo principal para comenzar
